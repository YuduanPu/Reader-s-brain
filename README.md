# Reader-s-brain
A writing skill for Claude that revises academic prose to reduce the reader's cognitive load, with a domain override layer for economics and quantitative social science.The base layer encodes a cognitive-science model of reading organized around five principles (Coherence, Continuity, Clarity, Concision, Cadence), derived from Yellowlees Douglas, Writing for the Reader's Brain: A Science-Based Guide (Cambridge University Press, 2025).The economics override layer (references/economics-overrides.md) adapts those rules for empirical and applied-theory work where prose carries epistemic content, not just processing cost. It protects causal-language calibration (matching verbs to the identification strategy), distinguishes load-bearing hedges from filler, replaces the "bury limitations" rule with field-appropriate handling of threats to identification, encodes the economics introduction genre, and enforces precise reporting of null results. When the two layers conflict, the override wins.Use it for drafting or revising manuscripts, abstracts, introductions, discussions, grant applications, and theses. Not for casual writing, marketing copy, or code.
